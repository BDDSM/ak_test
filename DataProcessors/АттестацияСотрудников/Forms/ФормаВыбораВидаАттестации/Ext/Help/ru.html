Форма предназначена для возможности выбора актуальной аттестации для прохождения тестирования,  а также предоставляет информацию о плановых датах прохождения следующих аттестаций.

ГЛАВНОЕ МЕНЮ
    • Начать аттестацию - Начать прохождение тестирования
    • Печать протокола - Печать протокола по Охране труда (других протоколов на текущий момент не предусмотрено)   

ТАБЛИЧНАЯ ЧАСТЬ
    • Организация - Организация в рамках которой проводится тестирование (указывается на форме аутентификации)
    • Вид аттестации - Вид аттестации 
    • Периодичность - Частота прохождения аттестации (Месяц, Год, Единоразово). 
    • Вопросов - Количество вопросов в текущей аттестации.
    • Минут - Количество минут, выделенных для прохождения тестирования.
    • Ошибок -Максимально возможное количество допущенных ошибок. 
    • Плановая дата с - Дата, начиная с которой можно пройти аттестацию.
    • Плановая дата по - Плановая дата аттестации.

ПРИМЕЧАНИЕ:
Если сотрудник проходит тестирование по определенному виду аттестации впервые, то в колонках «Плановая дата с» и «Плановая дата по» будет отображена надпись «Любое время». После успешного прохождения тестирования, «плановые» даты аттестаций будут заполнятся автоматически, согласно настроек вида аттестации.
