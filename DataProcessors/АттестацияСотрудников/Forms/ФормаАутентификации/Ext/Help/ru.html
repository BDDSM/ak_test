Форма аутентификации предназначена для контроля актуальность введенных пользователями данных, а также предоставления возможности перейти в раздел выбора аттестации для прохождения тестирования.

    • Организация - Тестирование могут проходить только сотрудники, принятые на работу в указанную организацию.
    • Сотрудник - Физическое лицо сотрудника, принятого на работу в вышеуказанную организацию . В форме выбора доступны только физические лица с признаком «Активный».
    • Пароль - Для пользователей имеющих именной бейдж(продавцы, кассиры) необходимо водить свой личный код бейджа. Система автоматически заполнит* поля Организация и Сотрудник.

* Если по введенному коду бейджа системе не удалось определить сотрудника (или автоматически заполненные данные по сотруднику/организации некорректны) следует воспользоваться стандартным паролем входа в подсистему - 1111.
