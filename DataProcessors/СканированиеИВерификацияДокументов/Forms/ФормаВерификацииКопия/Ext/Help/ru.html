Форма верификации

Эта форма открывается по нажатию на кнопку Верифицировать на основной форме обработки.

Тут отображаются атрибуты первого документа с закладки Сканирование. Форма верификации состоит из двух областей: реквизиты и табличная часть.

Реквизиты содержат исходные (взятые из результатов распознавания документов) и итоговые значения из базы. Исходные значения являются нередактируемыми, итоговые могут быть отредактированы пользователем. В табличной части документа строки с исходными значениями выделены бежевым цветом.

Поля могут быть помечены разными цветами:

  • красный - значение не найдено в базе;
  • зеленый - значение отличается от исходного;
  • розовый - необходимо сверить суммы;
  • оранжевый - единица измерения не сопадает с исходной.

На окне верификации располагаются следующие кнопки:

  • Добавить организацию (Создание организации) - добавление в справочник организаций новой организации; 
  • Добавить контрагента (Создание контрагента) - добавление в справочник контрагентов нового контрагента;
  • Открыть словарь соответствий (Словарь соответствий) - позволяет редактировать словарь соответствий;
  • Обновить полнотекстовый индекс - используется после добавления в справочник нового элемента (например, контрагента);
  • Работа с контактными данными - позволяет заменить адрес организации или контрагента в базе;
  • Выполнить (Запись и проведение документа) - в системе 1С будет создан документ на поступление товаров и услуг. Для того чтобы провести документ, необходимо нажать кнопку Ок;
  • Удалить документ - документ будет удален с закладки Сканирование. В окне верификации будет открыт следующий по порядку документ с закладки Сканирование, у которого стоит галочка в поле Экспортировать; 
  • Пропустить документ - на закладке Сканирование значение поля Экспортировать изменится на Нет (галочка будет снята). В окне верификации будет открыт следующий по порядку документ с закладки Сканирование, у которого стоит галочка в поле Экспортировать;
  • Закрыть - окно верификации закроется.

Проверка правильности распознавания

Подбор похожих значений

Подбор похожих значений осуществляется следующим образом:

  1. На этапе верификации выделить ячейку, помеченную красным цветом, и дважды кликнуть по ячейке. Появляются функциональные кнопки:
    • Выбрать из списка - открывает список наиболее похожих значений;
    • Открыть - открывает карточку объекта (организации, контрагента, товара, услуги);
    • Очистить - удаляет все данные, занесенные в ячейку (кнопка не доступна для табличной части документа).
  2. Нажать на кнопку Выбрать из списка. Появится список наиболее похожих значений из существующих в базе. Слева отображается процент схожести.
  3. Выбрать необходимое значение. Оно заменит исходное значение в ячейке, ячейка будет выделена зеленым.
  4. При выборе команды Выбрать значение из справочника откроется справочник Номенклатура.
  5. Выбрать необходимое значение из справочника. Ячейка пометится зеленым цветом.
  6. При нажатии правой кнопкой мыши по ячейке появляется контекстное меню, включающее сдедующие кнопки:
    • Подобрать рекомендуемое значение - открывает список подходящей номенклатуры, из которого можно выбрать необходимое значение;
    • Вернуть исходное значение - возвращает значение, которое было распознано изначально;
    • Создать номенклатуру (Создание номенклатуры) - позволяет создать новое значение в справочник Номенклатура (кнопка доступна только для табличной части документа).

Интеллектуальный подбор данных

Если на этапе верификации для какой-либо ячейки не было найдено значение в базе (ячейка выделена красным цветом), можно воспользоваться интеллектуальным подбором данных.

Для этого необходимо:

  1. Выбрать необходимую ячейку, выделенную красным цветом, и дважды щелкнуть левой кнопкой мыши. Появится возможность редактирования ячейки.
  2. Начать набирать значение в выделенной ячейке.
  3. Если значение найдено в базе, оно автоматически подставится в ячейку.
  4. Если значение не было найдено в базе, его можно добавить, нажав кнопку:
    • Добавить организацию (Создание организации) - для добавления организации в соответствующий справочник. После этого необходимо нажать кнопку Обновить полнотекстовый индекс;
    • Добавить контрагента (Создание контрагента) - для добавления контрагента в справочник Контрагентов. После этого необходимо нажать кнопку Обновить полнотекстовый индекс;
    • Создать номенклатуру (Создание номенклатуры) - для добавления наименования товара или услуги в справочник Номенклатура. Созданное наименование появится в табличной части документа в соответствующей ячейке.

Словарь соответствий

При обработке табличной части документа на этапе верификации система ищет в номенклатуре точное совпадение с учетом зависимых полей. В случае если значение не найдено, оно ищется в словаре соответствий.

Для редактирования словаря соответствий необходимо нажать на кнопку Открыть словарь соответствий.

Добавление новой записи в табличную часть

Чтобы создать новую запись в табличной части, необходимо выполнить следующие действия:

  1. В меню над табличной частью нажать кнопку Добавить. В конце таблицы добавятся две новые строки. Одна строка - исходное значение, другая строка - итоговое значение. Строка исходных значений является нередактируемой.
  2. Дважды щелкнуть по каждой ячейке новой строки, заполнить их значениями, введенными с клавиатуры или выбранными из справочника. Строка будет выделена зеленым цветом.

Создание номенклатуры

Номенклатуру можно создать вручную или автоматически.

  1. Чтобы создать номенклатуру вручную, необходимо выполнить следующие действия:
    1. В табличной части выделить ячейку колонки Номенклатура, помеченную красным цветом
    2. В выбранной ячейке ввести название нового товара или услуги. Нажать кнопку Создать номенклатуру (справа над табличной частью документа) или щелкнуть правой кнопкой мыши по ячейке и вызвать контекстное меню, в котором выбрать пункт Создать номенклатуру. Откроется окно создания новой номенклатуры.
    3. Заполнить открывшееся окно создания новой номенклатуры
    4. Нажать кнопку OK. В справочнике номенклатуры будет создана новая номенклатура. В табличной части выделенная ранее ячейка помечена зеленым цветом.
  2. Чтобы создать номенклатуру автоматически, необходимо выполнить следующие действия:
    1. На стадии верификации нажать кнопку Выполнить.
    2. В появившемся диалоговом окне нажать Да. В системе 1С будет создан документ. Все строки табличной части, наименования которых были помечены красным, будут созданы в справочнике номенклатуры.

Пересчет сумм

Пересчет сумм осуществляется следующим образом:

  1. В окне верификации отметить поле Всегда автоматически пересчитывать суммы в табличной части, в противном случае, при каждом пересчете будет всплывать диалоговое окно с потверждением пересчета.
  2. Изменить значение ячейки в колонке Количество/Цена/Сумма/Ставка НДС.
  3. Нажать клавишу Enter. Значения всех последующих ячеек строки пересчитаются в соответствии с внесенными изменениями. Все измененные вычисляемые ячейки пометятся зеленым цветом.
  4. Для ячеек, выделенных красным цветом, можно вызвать правой кнопкой мыши контекстное меню и выбрать пункт Подобрать рекомендуемое значение. Далее система вычислит необходимое значение и предложит его подставить. Измененная ячейка будет помечена зеленым цветом, а значения всех последующих ячеек автоматически пересчитаются.
  5. В модуле есть возможность сравнить исходное и расчетное значение суммы, как с учетом НДС, так и без учета НДС. Исходное значение основывается на исходных, распознанных значениях. Величина расчетного значения зависит от внесенных изменений.
