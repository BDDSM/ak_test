Обработка предназначена для формирования графика выходов в магазины для внештатных сотрудников. 

ВНИМАНИЕ!
При работе через Web интерфейс в браузере Google Chromе отмечена некорректная работа некоторых элементов управления 1С (полосы прокрутки, активация выделенных областей и прочее).
С выходом новых версий платфомы 1С и/или обновлений к браузеру Google Chromе ситуация может измениться.

На текущий момент рекомендуемые для работы браузеры: Mozilla Firefox, Opera, Safari, Internet Еxplorer.

 

ОПИСАНИЕ ЭЛЕМЕНТОВ

  1. ОБЛАСТЬ ГЛАВНОЕ МЕНЮ
    • Обновить – Обновление графика внештатных сотрудников.
    • Печать – Печать графика внештатных сотрудников.
    • Печать бейджей – Печать бейджей внештатных сотрудников (Примечание: Бейдж можно распечатать, только если по сотруднику установлен график на дату печати).
    • Отборы – Кнопка открытия дополнительных отборов (см. 2. ОБЛАСТЬ ОТБОРОВ).
    • Журнал учета – Переход в обработку «Журнал учета внештатных сотрудников».

  2. ОБЛАСТЬ ОТБОРОВ
    • Период – Период формирования графика.
    • Контрагент – Отбор по доступному контрагенту.
    • Магазин – Отбор по структурной единице.
    • Помощник – Отбор по помощнику.
    • Менеджер – Отбор по менеджеру (привязка устанавливается в карточке Внештатного сотрудника).
    • Должность – Отбор по должности (привязка устанавливается в карточке Внештатного сотрудника).
    • с.м. – Отбор по признаку «Сотрудник мобилен» (привязка устанавливается в карточке Внештатного сотрудника).

  3. ИНФОРМАЦИЯ О СОТРУДНИКЕ
    В области отображается информация о «текущем сотруднике», коде и телефоне сотрудника. Кнопка «Открыть» - открытие карточки «Внештатного сотрудника».

  4. МЕНЮ РЕДАКТИРОВАНИЯ ГРАФИКА
    Состав меню меняется в зависимости от наполненности основной области.
    • Добавить – Кнопка добавления сотрудника в график.

  5. ТАБЛИЧНАЯ ЧАСТЬ ТОРГОВЫХ ТОЧЕК
    Предназначена для группового редактирования графика «текущего сотрудника».
    (присутствует возможность использования клавиш Ctrl и Shift для выделения и добавления/редактирования нескольких строк табличной части).

  6. ОСНОВНАЯ ОБЛАСТЬ
    Область отображения графика внештатных сотрудников.

ФОРМИРОВАНИЕ ГРАФИКА

Для добавления сотрудника в "График работы внештатных сотрудников" следует придерживаться схемы:

    1. Установить период.
    2. При необходимости установить дополнительные отборы.
    3. Построить график (кнопка «Обновить»).
    4. Добавить сотрудника в график (кнопка «Добавить»).
    5. Выделить строку (или несколько строк с помощью клавиш Ctrl и Shift) в табличной части «Торговых точек» и по «двойному клику» (или кнопке «Изменить строку») открыть окно добавления Торговой точки и кол-ва часов:



Колонки формы добавления/редактирования торговых точек и кол-ва часов
(доступность колонок меняется в зависимости от указанного "Вида выхода"):

    • Вид – Выход, Отсутствие, Тренинг 
    • Тип – Дневной, Ночной (доступно только для вида «выход»).
    • Торговая точка – Структурная единица (доступно только для вида «выход»).
    • Кол. час. – Количество часов (доступно только для вида «выход»). 
    • Значение – Причина отсутствия (доступно только для вида «Отсутствие») или название тренинга (доступно только для вида «Тренинг»).
