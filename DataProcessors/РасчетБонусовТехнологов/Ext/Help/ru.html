1. Задвоение
Бонус = Процент_от_маржи * Маржа_характеристики * Коэф_задвоения * Коэф_включения_характеристики * Коэф_включения_холдинга

Где
Процент_от_маржи
Считается как 0,1% (см. вкладку Настройки.) от маржи характеристики.

Маржа_характеристики
Маржа считается как разница выручки (с учетом скидок) по характеристики (уменьшенная на НДС) и произведения количества продонного на себестоимость характеристики.

Коэф_задвоения
см. вкладку Настройки. Изначально берем 0.5 для не задвоенных товаров.
Товар считается задвоенным у технолога (рассчитывается только по производителям каждого из технологов), если есть минимум два производителя с долей выручки больше 5% (5% тоже вынести параметром в справочник настроек отчета).
Задвоения считать с учетом справочника "Аналоги номенклатуры". Товары попадающие в одну группу аналогов, даже без признака Полностью взаимозаменяемый товар считаются как один id_tov.
При этом могут быть товары-исключения, которые не требуют задвоения внутри технолога и для них коэффициент равен 1 при любых условиях.
Такие товары находятся в отдельном справочнике см. вкладку Настройки.
Если дата первой поставки товара (с учетом аналогов) + 3 месяца меньше чем начало расчетного периода, тогда Коэф_задвоения = 1, даже для не задвоенных товаров.
Есть справочник исключительных новинок, где вручную можно указать любой товар и дату первого прихода. см. вкладку Настройки.

Коэф_включения_характеристики
Считается по каждой характеристике товара. Коэффичиент зависит от количества включений. Вывести в справочник, с возможность указания для каждого количетсва включений, свой коэффициент.
В первой версии должны быть следующие значения
0 включений коэффициент равен 1
1 включение коэф. = 0.5
2 включения коэф. = 0.25
3 и более включения коэф. = 0
включения считаются по регистру Обращения покупателей. Попадают обращения у котоырх ТипСписанийИВозвратов = '01 Включение' или ТипОбращенийПокупателей = 01 Включение.
При этом Статус жалобы либо Подтверждена, либо В работе. И при этом Тип включения не равен "Нарушение технологии производства".

Коэф_включения_холдинга
считаются включения (условия для включений совпадают с Коэф_включения_характеристики) по всем характеристикам, относящимся к однму холдингу (из справочника Контрагенты).
Этот коэффициент - см. вкладку Настройки.
количество включений по холдингу считать как сумму всех включений деленую на количество продаваемых характеристик у холдинга за расчетный период.
если это расчетное кол-во больше значения из справочника, тогда обнуляются все характеристики холдинга.

2. Новинки
Бонус за новинку считается только по товарам новинкам (с разбивкой по технологам  с разбивкой на первый и второй месяц продаж следующим образом:
Выручка * Процент

где
Процент - параметр из справочника настроек расчета. Начальное значение 0,5%.
Выручка - сумма продаж с учетом всех скидок.

К новинкам относятся товары у которых выполняются следующие условия:
1. Берутся товары с учетом аналогов из справочника Аналоги номенклатуры (без учета полной взаимозаменяемости), без разбивки по технологам.
У этих товаров 30-й (бонус за первый месяц) или 60-й день (бонус за второй месяц) от первой поставки должен попадать в расчетный период.
Есть справочник, в котором любому товару можно вручную указать дату первой поставки см. вкладку Настройки. Для таких товаров-исключений дату первой поставки брать из справочника.
2. Средняя выручка товара за неделю с одного магазина должна быть не менее 700р (см. вкладку Настройки.).
или среднее количество проданного товара с одного магазина должно быть не менее 7 шт (см. вкладку Настройки.).
Для первого месяца средние продажи берутся с 24го по 30й день продаж с момента первой поставки.
Для второго месяца средние продажи берутся с 54 по 60й день продаж с момента первой поставки.
Количество магазинов при расчете средних продаж брать только то, где товар был за рассчетную неделю (с 24го по 30й или с 54го по 60й день). Были либо поставки, либо остатки, либо продажи (поставка + продажа + остаток > 0)

3. Снижение цены
Считается за первый месяц продаж по аналогии с бонусом за новинки.
Только берется товар+характеристика из отдельного регистра"Товары со сниженными ценам" см. вкладку Настройки.
В этом справочнике надо указать товар+характеристика и дату старта продаж. К этому справочнику не применяется проверка на средние продажи (минимум 700р или 7шт с магазина).
