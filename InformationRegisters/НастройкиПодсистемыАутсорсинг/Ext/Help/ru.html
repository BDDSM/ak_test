Регистр предназначен для хранения настрое подсистемы, а также значений "по умолчанию", которые используются при формировании документа "Заявка на услуги/материалы" из обработки "Журнала учета внештатных сотрудников (закладка "Заявка на услуги").

    • Использовать - Признак использования настройки

    • Основной ЦФО - ЦФО по умолчанию

    • Основной акцептант - Пользователь, обладающий правами на "Акцепт" заявки

    • НазначениеПлатежа - Текст назначения платежа (документ Заявка на услуги/материалы → Оплаты → Назначение платежа). Текст формируется динамически (доступны параметры: #Должность, #Период, #НомерДоговора)

    • Настройка "Дата платежа" - Недоступно для изменения. Дата платежа (документ Заявка на услуги/материалы → Оплаты → Дата платежа) устанавливается как ближайший от текущий даты рабочий (согласно Регламентированного производственного календаря) «Вторник» или «Четверг» (включительно).

    • Логирование ошибок импорта (Фронтол) - Запись ошибок в журнал регистрации (возникающих при импорте из Фронтол).

    • Группа контрагентов - Группа контрагентов предоставляющих «аутсорс» услуги.
