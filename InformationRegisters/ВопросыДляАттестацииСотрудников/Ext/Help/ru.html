Регистр сведений хранит информацию о вопросах и ответах для тестирования в разрезе «Организации» и «Вида аттестации»
    • Организация - Организация в рамках которой будет проводится аттестация
    • Вид аттестации - Вид аттестации
    • Вопрос - Поле вопроса может содержать только текст.
    • Ответ1...Ответ6 - Шесть возможных вариантов ответов*.
    • Правильный ответ - Номер правильного ответа.

*Варианты ответов следует заполнять по порядку, поскольку при заполнении, например, только 1-го и 3-го варианта ответа, у пользователя при прохождении теста помимо заполненных вариантов 1 и 3, будет также отображен «пустой» 2-й вариант. Выбрать «пустой» вариант пользователь не сможет, но отображение заполненного подобным образом вопроса будет выглядить «некорректным».

Кнопки редактирования картинки (доступны, если в указанном Виде аттестации установлен признак «Использовать картинки вопросов»):
    • Добавить картинку - Добавление картинки (хранится в справочнике «Файлы аттестации»).
    • Удалить картинку - Непосредственное удаление картинки из справочника «Файлы аттестации».
