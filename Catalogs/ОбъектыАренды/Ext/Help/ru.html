Справочник "Объекты аренды"

 

Справочник предназначен для хранения информации об объектах аренды.

Реквизиты справочника:

  • Наименование. Наименование объекта. При создании нового элемента справочника по умолчанию ему присваивается составное наименование. Оно состоит из названия группы, в которой находится данный элемент, и кода нового элемента.
  • Группа. Ссылка на группу справочника, в которую входит элемент. При ручном изменении группы происходит переброска элемента из одной группы в другую.
  • Тип объекта. Один из типов объектов аренды, определенный в справочнике «Типы объектов аренды».
  • Здание и этаж. Расположение объекта аренды в помещении.
  • Адрес объекта. Адрес объекта аренды.
  • Площадь. Площадь объекта аренды (в кв.м.).
  • Расположение на плане. Привязка объекта аренды к конкретному графическому объекту, размещенному на рабочем листе таблицы MS Excel.
  • Описание объекта аренды. Текстовая информация об объекте аренды.
  • Не контролировать занятость. Признак контроля занятости помещения при заключении договора аренды. Если установить данный признак, то объект можно сдать в аренду одновременно нескольким арендаторам.
  • Не контролировать площадь. Признак контроля сдаваемой площади помещения при заключении договора аренды. Если установить данный признак, то можно сдать площадь по данному объекту (одному или нескольким арендаторам) в сумме превышающую площадь самого объекта недвижимости. Excel.

 

Таблица Элементы содержит список элементов инженерных систем объекта.

Для объектов аренды можно задать дополнительные аналитические признаки – свойства. Данная информация используется для отбора элементов номенклатуры или группировок в отчетах. Открыть свойства можно нажатием кнопки "Открыть свойства" на форме элемента.

В блоке "Собственник объекта" отображается информация о собственнике данного объекта недвижимости, о договоре с собственником и используется ли агентская схема при сдаче в аренду данного объекта. Данную информацию можно изменить в регистре сведений "Собственники объектов аренды", форма которого открывается по кнопке "Изменить".
