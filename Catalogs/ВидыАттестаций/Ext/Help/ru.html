Справочник предназначен для хранения информации об Аттестации и наряду со справочником "Организации" является основным разделителем подсистемы Аттестация персонала
Также в справочнике хранится инструкция для прохождения аттестации.

Реквизиты справочника:

    • Организация - Пройти аттестацию смогут только сотрудники, принятые на работу в указанную организацию.
    • Периодичность - частота прохождения аттестации (Месяц, Год, Единоразово).
    • К-во дней (до начала) - Показатель участвует в расчете даты (в случае указанной периодичности Месяц или Год) начиная с которой сотрудник может пройти аттестацию. Если сотрудник в период с "К-во дней (до начала)" до "Плановой даты" не проходит тестирование, то аттестация будет считаться "просроченной".    
    • Использовать картинки вопросов - При установленном флаге в регистр Вопросы для аттестации сотрудников при заполнении вопросов/ответов можно будет добавлять картинки.
    • Актуальная - Признак актуальности аттестации. В форме выбора аттестации перед прохождением отображаются только "Актуальные" элементы.
    • Имя файла - Имя добавленного в элемента файла с инструкцией. 
  
Инструкция:

   • Добавить - Добавление файла с инструкцией к аттестации (*.doc, *.docx).
   • Удалить - Удаление файла с инструкцией.
   • Открыть - Открытие файла.
