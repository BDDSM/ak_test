Справочник "Группы писем электронной почты"

Данный справочник предназначен для хранения групп писем электронной почты для каждой учетной записи. При создании нового элеменита справочника Учетные записи электронной почты в текущем справочнике создаются четыре элемента — группы электронных писем по умолчанию для новой учетной записи:

  • Входящие
  • Исходящие
  • Черновики
  • Удаленные

Справочник является иерархическим: при необходимости пользователь может создавать новые группы и подчинять их друг другу.
