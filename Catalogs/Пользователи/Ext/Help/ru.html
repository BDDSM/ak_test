Справочник «Пользователи»

Справочник «Пользователи» предназначен для хранения списка пользователей. В основном это пользователи, работающие с конфигурацией (пользователи ИБ).

Идентификация пользователя ИБ с пользователем справочника выполняется по соответствию имени пользователя ИБ имени пользователя справочника.

Интерфейс

Пользователям можно назначить Профиль, указать Физическое лицо, которое соответствует данному пользователю.

Редактирование свойств пользователя ИБ выполняется на закладке "Свойства пользователя информационной базы".

Дополнительные сведения

Редактирование дополнительных сведений выполняется через подменю "Дополнительные сведения".

Дополнительные сведения доступны только в обычном приложении.

  • Настройки пользователя - позволяет изменить настройки пользователя
  • Контактная информация - позволяет изменить контактную информацию, которая используется при работе пользователя с клиентами и при работе со встроенной электронной почтой
  • Группы пользователя - позволяет изменить группы, в которые входит пользователь
  • Дополнительные права - позволяет изменить дополнительные права пользователя

Создание новых пользователей

Только Администратор пользователей может создавать, изменять и удалять пользователей.

Создание пользователей ИБ

Создание пользователей ИБ можно выполнять в режиме конфигуратора или в режиме предприятия.

Управление свойствами пользелей ИБ в режиме предприятия является более предпочтительным, чем непосредственное управление пользователями через конфигуратор.

Полномочия пользователей

Полномочия пользователей определяются его ролями и дополнительными правами.

Полномочия можно назначать при помощи профилей.

Права доступа на уровне записей определяются группами пользователей, в которые входит пользователей.
