Подсистема «Аттестация персонала» включает объекты конфигурации обеспечивающие возможность поводить настройку, тестирование и аттестацию (с заданной периодичностью) сотрудников.


Основной состав подсистемы:

  ♦  Справочник Виды аттестаций - Наряду со справочником «Организации» является основным разрезом подсистемы.
  ♦  Справочник Файлы аттестаций - Хранение файлов подсистемы.
 
  ♦  Регистр сведений Настройки аттестации - Хранение настроек для прохождения тестирования.
  ♦  Регистр сведений Вопросы для аттестации сотрудников - Хранение информацию о вопросах и ответах для прохождения тестирования.
  ♦  Регистр сведений Итоги прохождения аттестации сотрудников - Хранение данных о результатах тестирований.

  ♦  Роль «Ответственный за аттестацию» - Предназначена для возможности настройки подсистемы, и установки признака «подтверждения» по итогам пройденных сотрудниками аттестаций.
  ♦  Роль «Пользователь аттестация» - Предоставляет возможность прохождения тестирования. 
 
Поддерживаемые варианты запуска:

  • Тонкий клиент (Управляемое приложение)
  • Толстый клиент (Управляемое приложение)
  • Web клиент
