Документ предназначен для формирования потребностей в новых сотрудниках и последующего обеспечения персоналом, принятым на работу в отделе кадров.

ОСНОВНАЯ СХЕМА РАБОТЫ

РЕКВИЗИТЫ ДОКУМЕНТА

    • Дата - Дата документа.
    • Номер - Номер документа.  
    • Организация - Организация в рамках которой будет проводится формирование и закрытие потребности.
    • ЦФО - Группа торговых точек в рамках которой будет проводится формирование и закрытие потребности.
    • Статус - Статус заявки (Новая, В работе, Обеспечена, Отменена). 
                   Статусы устанавливаются автоматически.
                   ВАЖНО: При полном обеспечении заявки (статус Обеспечена) документ смогут редактировать только пользователи с ролью "Полные права".
    • Город - Город (информационное поле).
    • Метро - Метро (информационное поле).   
    • Ответственный - Пользователь создавший документ.

ТАБЛИЧНАЯ ЧАСТЬ ПОТРЕБНОСТИ
(предназначена для формирования потребности в персонале Торговыми управляющими, или Помощниками ТУ)

    • Должность - Должности, в которых возникла потребность (можно выбирать только из списка существующих должностей, из реквизита Должность в карточках физ. лиц).
    • Количество - Количество сотрудников, которые должны обеспечить в отделе кадров.
    • Комментарий - Комментарий.

ТАБЛИЧНАЯ ЧАСТЬ ОБЕСПЕЧЕНИЕ
(предназначена для закрытия потребностей в персонале, сформированных Торговыми управляющими, или Помощниками ТУ)

    • Сотрудник - Сотрудник, которым закрывают выставленную потребность.
    • Должность - Должности, в которых возникла потребность (заполняется автоматически)
    • Комментарий - Комментарий.
